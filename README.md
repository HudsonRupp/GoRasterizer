# GoRasterizer

3D Rasterizer from scratch in Go.

## Execution 

If on windows:
```bash
export GOOS=windows
```

To run:
```bash
go run . obj/teapot.obj
```

## Controls
- WASD to move camera forward/back + left/right, Q/E to move camera up/down 
- Use arrow keys to change the direction of the camera
